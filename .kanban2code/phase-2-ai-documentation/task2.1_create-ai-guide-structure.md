# Task 2.1: Create AI Guide Structure

## Goal

Plan the structure of the AI guide document that will teach AIs how to work with Kanban2Code.

## Background

AIs need comprehensive, clear documentation to understand how to create tasks, use templates, and follow the Kanban2Code workflow correctly.

## Scope

- Define how Kanban2Code works (brief overview)
- Specify file format requirements
- Document frontmatter schema with all fields
- Define stage definitions and transitions
- Explain the context system
- Document user preferences and conventions
- Plan examples of well-formed tasks
