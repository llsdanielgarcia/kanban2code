# Migration Guide for Existing Users

## Goal
Create migration guide for users upgrading from template-based to agent-driven workflow.

## Definition of Done
- [ ] MIGRATION.md or section in README created
- [ ] Steps to migrate existing workspaces documented
- [ ] Decision on existing `_templates/` folders (delete or ignore)

## Context
This task creates documentation to help existing users migrate to the new agent-driven workflow, including what happens to their existing template files and how to adapt to the new way of working.
