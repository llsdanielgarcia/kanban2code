# Task 4.3: Create Audit File Template

## Goal

Create a template for `audit-phase#.md` files that track phase completion and quality metrics.

## Background

Each phase needs an audit file to document the review process, track quality metrics, and provide sign-off.

## Scope

- Phase summary section
- Tasks reviewed list
- Code quality evaluation (0-10 scale)
- Test coverage status
- Issues found
- Recommendations
- Sign-off section

## Files to Create

- `_templates/context/audit-phase.md` - Audit file template

## Acceptance Criteria

- [ ] Phase summary section is included
- [ ] Tasks reviewed are tracked
- [ ] Code quality evaluation scale is defined
- [ ] Test coverage status is captured
- [ ] Issues found section is included
- [ ] Recommendations are documented
- [ ] Sign-off section is provided
- [ ] Template is structured for consistency

## Testing Requirements

Verify the template can be used to create valid audit files.

## Notes

Audit files serve as quality gates and should provide clear evidence of phase completion.
