# Task 4.2: Create Phase Context Template

## Goal

Create a template for per-phase context files that track progress and provide context for ongoing work.

## Background

Each phase needs its own context file to track objectives, completed tasks, current state, and pending decisions.

## Scope

- Phase objectives section
- Completed tasks summary
- Current state of implementation
- Pending decisions
- Context for next tasks
- Update instructions

## Files to Create

- `_templates/context/phase-context.md` - Phase context template

## Acceptance Criteria

- [ ] Phase objectives section is included
- [ ] Completed tasks summary is tracked
- [ ] Current state section documents progress
- [ ] Pending decisions are captured
- [ ] Context for next tasks is provided
- [ ] Update instructions are clear
- [ ] Template is lightweight and focused

## Testing Requirements

Verify the template can be used to create valid phase context files.

## Notes

Phase context files should be updated after each task completion to maintain continuity.
