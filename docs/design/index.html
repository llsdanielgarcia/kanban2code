Design Implementation Notes

  • VS Code Integration: All colors use --vscode-* CSS variables with dark mode fallbacks for browser preview.
  • Glassmorphic Style: Uses backdrop-filter: blur() and subtle gradients. May need fallbacks for older browsers.
  • Stage Colors: Each stage has a primary color, background (10% opacity), and border (20% opacity) variant.
  • Responsive: Board columns are fixed-width (280px) with horizontal scroll. Swimlanes stack vertically.
  • Interactions: Hover states reveal card actions. Drag-and-drop zones highlighted on drag.
  • Accessibility: Focus states use VS Code's focus border color. ARIA labels to be added during React implementation.
