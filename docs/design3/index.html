Design Implementation Notes

  • Palette: Midnight base with neon mint + glacier blue accents; stage/tag colors updated to feel consistent in dark mode.
  • Typography: Sora + Space Grotesk pairing for a cleaner, futuristic tone with tighter letter-spacing on labels.
  • Surfaces: Matte glass panels with softer shadows and wider radii; gradient edges replace the older frosted blur heavy look.
  • Layouts: Sticky board header, 300px columns, and refreshed swimlane shells using the same tokens for consistency.
  • Interactions: Stage chips, action buttons, and cards lift on hover; drag/drop uses the neon mint + blue focus cues.
  • Accessibility: Focus rings track the accent blue; keep ARIA labels when wiring into React so keyboard users can navigate.
